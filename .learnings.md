# Learnings: Mistral Vibe Refactor

## Project Structure
- core engine: `vibe/core`
- CLI TUI: `vibe/cli`
- Protocol Bridge: `vibe/acp`
- Setup Wizards: `vibe/setup`

## Conventions
- Use `uv run` for all commands.
- Python 3.12+ features: `match/case`, modern type hints (e.g., `list[str]`), `pathlib.Path`.
- Absolute imports: `from vibe.core.x import ...`.
- Declarative, minimalist code.
- No relative imports.
- No inline type ignores.
