# Design System Strategy

## Overview
This document outlines the design principles and system architecture for the Open Design Template Factory.

## Tokens
Foundational design tokens are stored in `design-system/tokens/`.
- Colors
- Typography
- Spacing
- Radius
- Shadows

## Patterns
Common layout patterns are stored in `design-system/patterns/`.

## Components
Reusable UI components are stored in `design-system/components/`.

## Governance
All design changes must be verified against the `design-evaluator.md`.
