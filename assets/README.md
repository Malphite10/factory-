# Asset Factory Rules

## Subdirectories
- `thumbnails/`: 600x450 PNG.
- `previews/`: Full-page high-res screenshots.
- `social/`: OG images (1200x630).
- `marketplace/`: Specific marketplace-ready bundles.
- `screenshots/`: Multi-device feature highlights.

## Optimization
- All images must be ran through `imagemin`.
- WebP preferred for previews.
