# Agent Operating Modes

modes:
  - ideation: Conceptualizing the workflow and agent roles.
  - research: Market and trend analysis.
  - production: Active execution of agent tasks.
  - optimization: Refining agent logic and handoff speed.
  - maintenance: Updating dependencies and schemas.
  - release: Final QA and deployment.

## Performance Guidelines
- All templates must score > 90 on Lighthouse.
- CLS must be < 0.1.
- LCP must be < 2.5s.
